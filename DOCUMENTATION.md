# Pleasure Control - Финальная документация

## 📋 Обзор проекта

**Pleasure Control** - профессиональное веб-приложение для управления интимными устройствами через протокол Buttplug.io. Приложение полностью переработано с современным UI/UX, расширенным функционалом и персонализацией.

## ✨ Реализованные функции

### Фаза 1: UI/UX Модернизация ✅
- ✅ Темная тема с фиолетово-розовыми акцентами
- ✅ Переключатель темы (светлая/темная)
- ✅ Анимированный splash screen
- ✅ Ребрендинг в "Pleasure Control"

## 🎨 Структура навигации

1. **Devices** - Подключение и управление устройствами
2. **Manual** - Ручное управление слайдерами
3. **Patterns** - Библиотека паттернов и редактор
4. **Modes** - Автоматические режимы (Surprise, Edge, Sync)
5. **Games** - Интерактивные мини-игры
6. **Audio** - Аудио-реактивное управление
7. **Settings** - Настройки и статистика

## 🛠️ Технический стек

- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Device Control**: Buttplug.io v4.0.2
- **Build**: Vite 7
- **State Management**: React Hooks
- **Storage**: localStorage

## 🚀 Запуск приложения

```bash
cd app

npm run dev
```

Приложение откроется на `http://localhost:5173`

## 🔧 Сборка для продакшена

## 🎨 Кастомизация

### Изменение акцентного цвета
Перейдите в Settings → Accent Color и выберите один из 6 цветов:
- Purple (по умолчанию)
- Pink
- Blue
- Green
- Orange
- Red

### Настройка лимитов
Settings → Safety & Defaults:
- Maximum Intensity (30-100%)
- Default Intensity
- Auto-Stop Timeout (5-60 минут)

## 📖 Использование

1. **Подключение устройства**:
   - Devices → Connect to Intiface
   - Scan Devices
   - Дождитесь обнаружения устройства

2. **Ручное управление**:
   - Manual → выберите устройство
   - Используйте слайдеры для управления

3. **Использование паттернов**:
   - Patterns → выберите пресет или создайте свой
   - Play для запуска

4. **Игры**:
   - Games → выберите игру
   - Настройте параметры
   - Start для запуска

5. **Просмотр статистики**:
   - Settings → Statistics
   - Экспорт данных при необходимости

## 🐛 Известные ограничения

- Требуется запущенный Intiface Engine на порту 12345
- WebBluetooth может не работать в некоторых браузерах
- Некоторые устройства не поддерживают мониторинг батареи
- Аудио-реактивный режим требует разрешения на микрофон
